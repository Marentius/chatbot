# Almesus

Nettside for Almesus-festivalen i Jelsnes. Inneholder aftermovie, bildegalleri og festivalinformasjon - bygget med TypeScript og React.

### Aftermovie

Videoavspiller med høydepunkter fra Almesus 2024

### Bildegalleri

Fotogalleri med bilder fra festivalen i responsivt grid

### Lokasjon

Kart og praktisk informasjon for festivalbesøkende

## Funksjoner

### Aftermovie

Integrert videoavspiller som viser høydepunkter fra festivalen med aftermovie fra Almesus 2024.

### Bildegalleri

Responsivt fotogalleri med bilder fra festivalen, organisert i et oversiktlig grid-layout.

### Festivalinformasjon

Informasjon om festivalen med lokasjon, kart og praktiske detaljer for besøkende.

### Responsivt design

Fargerikt gradient-design med grønn-til-gul bakgrunn og rosa navigasjon, optimalisert for alle skjermstørrelser.
