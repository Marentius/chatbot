# FileConverter

Et universelt filkonverteringsverktøy med batch-støtte og OCR-funksjonalitet. Tilbyr både CLI- og desktop GUI-grensesnitt bygget med Tauri og React.

### CLI

Kommandolinjeverktøy for kraftbrukere og automatisering. Perfekt for batch-konvertering og integrasjon i eksisterende arbeidsflyter og scripts.

### Desktop GUI

Tauri + React-basert desktop-applikasjon for et brukervennlig grafisk grensesnitt. Cross-platform med native ytelse og lavt minneforbruk.

## Funksjoner

### Bildekonvertering

Konverter mellom alle populære bildeformater med ImageMagick - PNG, JPG, WebP, SVG og mer.

### Dokumentkonvertering

Konverter Office-dokumenter, PDF-er og markup-filer med LibreOffice og Pandoc.

### OCR-funksjonalitet

Trekk ut tekst fra bilder og PDF-er med Tesseract og ocrmypdf for søkbare dokumenter.

### Batch-konvertering

Konverter mange filer samtidig med presets for optimaliserte innstillinger.

### CLI-grensesnitt

Kraftig kommandolinjeverktøy for automatisering og scripting av filkonverteringer.

### Desktop-applikasjon

Tauri-basert GUI med React for et brukervennlig grafisk grensesnitt på alle plattformer.
